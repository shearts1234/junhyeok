메뉴 결정 장애를 해결하는 '오늘 뭐 먹지?' 서비스 소개

'오늘 뭐 먹지?'는 매일 반복되는 식사 메뉴 고민(결정 장애)을 해결해 드리는 인공지능 기반 메뉴 추천 서비스입니다. 직장인 점심, 가족 저녁 식사 등 다양한 상황에서 100가지 이상의 인기 메뉴 중 당신의 입맛을 사로잡을 최고의 한 끼를 제안합니다.

주요 추천 기능 상세

  • 스마트 랜덤 메뉴 추천: 한식, 중식, 일식, 양식, 분식 등 카테고리별 검증된 메뉴를 무작위로 선정하여 선택의 고통을 덜어드립니다.
  • 고퀄리티 AI 이미지 실시간 생성: 구글의 최신 모델인 Gemini 3.1을 연동하여, 추천된 메뉴의 이미지를 텍스트가 아닌 시각적인 그림으로 즉시 보여줍니다.
  • 간편한 사용자 경험: 복잡한 설정 없이 버튼 하나로 오늘의 메뉴를 추천받을 수 있습니다.

자주 묻는 질문 (FAQ)

Q. 오늘 뭐 먹지 서비스는 무료인가요?

네, 기본적인 메뉴 추천 기능은 누구나 무료로 이용할 수 있습니다. 다만, AI 이미지 생성 기능을 사용하려면 본인의 Gemini API Key가 필요합니다.

Q. 추천되는 메뉴의 종류는 얼마나 되나요?

현재 약 100여 가지의 대중적인 메뉴 데이터베이스를 보유하고 있으며, 지속적으로 업데이트하고 있습니다.

Q. 개인정보(API 키)는 안전하게 보호되나요?

입력하신 API 키는 서버로 전송되지 않고 사용자의 브라우저 로컬 스토리지에만 안전하게 저장됩니다. 안심하고 사용하셔도 됩니다.
